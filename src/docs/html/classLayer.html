Detailed Description

Core class of the framework, interface for each layer. Each layer can be seen as a node in the computation graph as such, it has an input, it has to be capable of computing the output from the given input, and it has to be able to compute it's parameter gradients and also the gradient of error w.r.t. the input of this layer, which can then be backpropagated to previous layers.

Clearly this doesn't hold for InputLayer.
